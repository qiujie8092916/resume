Project

在 Strurs2 框架下以 MVC 设计 Java 为后台，以表格展现 mysql 数据库中流媒体文件的信息，以及对新的流媒体添加信息后上传。 用户登录后，通过设置可保存可分享的偏好、并可通过联合过滤搜索，对流媒体文件的相关属性进行查阅，可对单个文件下载、查询结果列表下载、文件信息修改等功能。 Check It

Highcharts 为框架， ajax 通过 php 后台查询 mysql 数据库，查询流媒体测试结果。 依次从 Package、Suite、Case、Method 缩小范围对不同版本，以折线图查看结果曲线，最后结果通过热力图显示每个版本上的不同 Method 的测试结果以及相关信息，通过链接跳转可看相关详细信息。 Check It

Bootstrap Table 为框架，Client 端通过 mysql 查询 AutomationCase 在固定平台的不同job的测试结果。 可通过搜索、过滤、日期查询等方式，展现每轮测试结果以及 Build type、Branch name 等不同相关属性。点击可跳转到每次测试结果的日志、状态以及一键 Restart。 Check It
